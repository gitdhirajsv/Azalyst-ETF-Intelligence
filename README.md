# Azalyst ETF Intelligence

This is a tool for managing ETF investments, featuring:

- Data analysis
- Automated reporting
- Portfolio management

## Installation

Run the script using:

```bash
python Azalyst_Spyder.bat
```

## Usage

After installation, execute the script as follows:

```bash
./Azalyst_Spyder.bat
```
